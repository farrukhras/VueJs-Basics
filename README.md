# VueJs Basics

Basics of VueJs are covered in this repo. The main contents of the Vue framework covered are:

1. The Vue Instance
2. Data and Methods in Vue
3. Data Binding
4. Events in Vue
   - _click_
   - _dblclick_
5. Keyboard Events
   - _keyup_
   - _keydown_
6. Event Modifiers
   - _enter_
   - _once_
7. Computed Properties
8. Conditionals
   - _v-if_
   - _v-else-if_
   - _v-else_
   - _v-show_
9. Looping in Vue using v-for
10. Refs
11. Multiple Vue Instances
12. Vue Components
13. Vue Directive
    - _v-model_
    - _v-on_
    - _v-bind _
    - _v-html_
    - _v-for_
    - _v-if/v-else-if/v-else_
    - _v-show_

#### There is also a mini Game (PunchBag Game) made using the above mentioned basics of Vue.
#### There are extensive comments to help you understand the concepts in detail and help you along the way.
